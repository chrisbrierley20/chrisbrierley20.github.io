Olympinomics - The Economic Impacts of the Olympic Games

Introduction

This project aims to investigate various macro and microeconomic impacts the Olympic games can have on host nations, competitors and firms alike. Ultimately, this results in hopefully deciding whether hosting or sponsoring the Olympics is an economically viable strategy on improving various economic indicators.

Challenges in data cleaning/analysis and how I overcame them

By far the greatest challenge I faced when cleaning data was with the Exchange Rate Index chart. This required taking daily exchange rate data of USD to 1 unit of local currency from FRED for the 5 previous currencies of Olympic hosts between 2000 and 2021 and converting to an index. I then created a separate Olympic index inspired by Goldman Sachs (2012) (https://www.goldmansachs.com/insights/archive/archive-pdfs/olympics-and-economics-.pdf) which follows the currency of an Olympic host from 4 years prior to the day of the closing ceremony before changing to the next currency. Furthermore, using daily data would make the chart almost illegible, so I had to use Excel filters to only include the closest date to the first of the month, thereby now only using monthly data.

There were also problems with scraping Wikipedia in Python for Olympic medal tables and number of athletes. This is because for each year, the columns in each table would differ considerably. Time was therefore spent manually inputting into Python which rows should be deleted from each unique table. Combining these finalised tables into one final spreadsheet was also particularly difficult when purely using Python, so I instead opted to use Excel to make my final, cleaned spreadsheet.
